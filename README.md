CLI Task Tracker на Python.

## Функции
- добавить задачу
- показать задачи
- отметить задачу выполненной
- удалить задачу
- сохранить задачи в JSON

## Запуск

```bash
python main.py
```

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск в браузере

```bash
python web.py
```

## API-документация

```bash
http://127.0.0.1:8000/docs
```

## Тесты

```bash
python -m unittest discover -s tests -v
```
