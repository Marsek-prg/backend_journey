# Backend Journey

Мой путь в backend-разработку с нуля.

## Первый проект
CLI Task Tracker на Python.

## Функции
- добавить задачу
- показать задачи
- отметить задачу выполненной
- удалить задачу
- сохранить задачи в JSON

## Запуск

```bash
python main.py
```

## Запуск в браузере

```bash
python web.py
```
